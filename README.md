## Установка

Установка зависимостей
`yarn`

Запуск локальный
`yarn dev`

Запуск билда
`yarn buid`

Typescript watcher
`yarn watch-ts`

Проверки
`yarn check-lint`
`yarn check-ts`
`yarn check-style`
